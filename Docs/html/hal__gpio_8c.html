Detailed Description

STM32F4 GPIO driver implementation for bare-metal HAL.

Provides GPIO control: initialization, alternate function setup, mode switching, digital read/write—all via direct register access. No CMSIS or STM32 HAL used.

Author: Steven Reynoso
