Detailed Description

High-level UART interface for STM32F411RE.

Provides simple UART communication functions including initialization, string transmission, and byte reception using polling (blocking mode). Built on top of direct register access to STM32F4 UART hardware.

Transmits each character in the given string one by one, waiting for the TXE (Transmit Empty) flag before sending the next character.
