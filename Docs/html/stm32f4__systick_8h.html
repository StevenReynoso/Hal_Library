Detailed Description

Register map and macros for SysTick timer on STM32F411RE.

This header defines the memory-mapped structure and bit masks for the Cortex-M4 SysTick timer. SysTick is a simple 24-bit timer integrated into the Cortex core, commonly used for periodic interrupts or delay generation in bare-metal systems.
