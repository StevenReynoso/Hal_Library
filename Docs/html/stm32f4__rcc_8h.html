Detailed Description

RCC (Reset and Clock Control) register map for STM32F4 series.

Defines the base address and register layout of the RCC peripheral, used for configuring clocks, enabling peripheral buses, and managing low-power/reset states.

This is a low-level, direct-register header with no runtime logic.
