Detailed Description

Timer driver for STM32F446RE (PWM and basic timer functions).

This file provides simple timer functions for:

  • Generating basic delays (1 Hz overflow)
  • Setting up PWM output (for motors, LEDs, etc.)

This is part of a custom STM32F4 HAL written from scratch with no STM HAL or CMSIS. All register access is direct and uses only official STM32 documentation.
