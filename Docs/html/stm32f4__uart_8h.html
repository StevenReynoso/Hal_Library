Detailed Description

Register map and macro definitions for UART/USART on STM32F411RE.

This header defines memory-mapped structures and constants for the STM32F4 UART and USART peripherals. UART is used for asynchronous serial communication between the MCU and devices like PCs, GPS modules, Bluetooth, or other microcontrollers.

Supported registers include control, status, baud rate, and data registers.
