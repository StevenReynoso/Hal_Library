Detailed Description

Public GPIO HAL API for STM32F4xx.

STM32F4 GPIO peripheral definitions and configuration structures.

This header provides high-level GPIO control functions for use with STM32F411RE microcontrollers in bare-metal applications.

All functions operate using the gpio_config_t structure and packed pin macros.

Tip: In VS Code, you can hide all comments using Ctrl + K, then Ctrl + 0.

This header provides:

  • Base address macros for all GPIO ports (A–H)
  • A compact pin representation using packed encoding
  • A register layout for direct register access
  • GPIO configuration structures and enums

Designed for use in bare-metal STM32F411RE projects without STM HAL or CMSIS.
