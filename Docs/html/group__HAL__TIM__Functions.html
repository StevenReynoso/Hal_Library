Initializes the specified timer to generate a 1 Hz overflow.

Sets up prescaler and ARR values to create a 1-second periodic timer overflow. Useful for software delay loops, polling, or LED blinking.

Parameters
timxTimer instance (e.g., TIM2, TIM3).
clk_hzTimer input clock frequency in Hz (APB1/APB2 dependent).
Note
Timer is started immediately. No interrupt or output configuration is done.

Initializes the specified timer to generate a 1 Hz overflow.

This sets up a basic timer using a fixed prescaler and a computed auto-reload value. When started, the timer will count up and overflow once every second.

Useful for:

  • Blinking an LED
  • Basic delay loops (without interrupts)
  • Timing functions where accuracy is not critical
Parameters
timxPointer to the TIMx peripheral (e.g., TIM2, TIM3).
clk_hzThe clock frequency driving the timer (in Hz).
Note
This function starts the timer immediately in upcounting mode.
Warning
Assumes a hardcoded prescaler of 1600.
