Detailed Description

Basic timer initialization interface for STM32F411RE.

This header exposes a helper function for initializing general-purpose timers to generate a 1 Hz timebase using prescaler and auto-reload configuration.

Initializes the specified timer to generate a 1 Hz timebase.

Configures the timer's prescaler and auto-reload register (ARR) so that it overflows every 1 second. This can be used for timekeeping, simple blinking, or periodic polling loops.

Parameters
timxPointer to the timer peripheral (e.g., TIM2, TIM3, etc.)
clk_hzInput clock frequency in Hz (usually equal to APB1/APB2 timer clock)
Note
This function only sets up the timer for basic up-counting operation. It does not enable interrupts or configure any output channels.

Initializes the specified timer to generate a 1 Hz timebase.

This function calculates a prescaler and auto-reload value so that the timer overflows every 1 second. It assumes the timer is driven by clk_hz, which should be the correct APB1 or APB2 timer clock depending on the TIM peripheral.

Parameters
timxPointer to the timer peripheral (e.g., TIM2, TIM3, etc.)
clk_hzThe timer input clock frequency in Hz.
Note
This function enables the timer but does not enable interrupts. You must call rcc_enable_tim() beforehand to ensure the clock is active.
