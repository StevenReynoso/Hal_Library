Detailed Description

SysTick timer implementation for STM32F411RE.

This file implements initialization and delay routines using the Cortex-M4 SysTick timer. Designed for simple blocking delays in microseconds or milliseconds. Suitable for low-frequency, low-complexity tasks or startup-time code in bare-metal applications.
