Detailed Description

Core HAL type definitions and centralized module includes for STM32F411RE.

This header includes all major HAL modules (GPIO, RCC, SysTick, TIM, UART, SPI) and defines basic types used across the HAL codebase. It serves as a common import point for user applications and higher-level drivers.
